  知枢 · 本机 agent（Web V1.0）
═══════════════════════════════════════════════════════

这是一个"自动整理你的文档、笔记、材料"的本地工作台。
所有数据都留在你这台电脑上；只有用户名和心跳会同步到云端。


┌─── 你需要准备的 ────────────────────────────────────┐
│                                                      │
│  1. Windows 10 或 11                                 │
│  2. Python 3.10 或更高版本                            │
│     如果没装：https://www.python.org/downloads/      │
│     安装时勾选 "Add Python to PATH"                  │
│  3. 来自管理员的三件事：                              │
│     ① 云端地址（http://...:8090）                    │
│     ② JWT 密钥（一长串字母数字）                       │
│     ③ 你的云端用户名                                  │
│                                                      │
└──────────────────────────────────────────────────────┘


┌─── 三步上手 ────────────────────────────────────────┐
│                                                      │
│  第 1 步：双击 setup.bat                             │
│           按提示输入云端地址、JWT 密钥、用户名         │
│           （大约 2-3 分钟，会自动装依赖）              │
│                                                      │
│  第 2 步：双击 start.bat                             │
│           窗口会弹出来显示日志；保持开着              │
│                                                      │
│  第 3 步：浏览器打开云端地址，用第 1 步填的用户名登录  │
│                                                      │
└──────────────────────────────────────────────────────┘


┌─── 你的数据在哪 ────────────────────────────────────┐
│                                                      │
│  C:\Users\你的Windows用户名\.handynotes\<云端用户名>\ │
│                                                      │
│  里面有：                                             │
│    wiki\        知识页（markdown，自动整理出来的）     │
│    raw\         原始文档归档                          │
│    data\        系统数据（活动日志、批次元数据）       │
│    config\      schema 和配置                         │
│    .env         你的 LLM API key（如果填了）          │
│                                                      │
│  整个目录可以备份、可以拷贝到别的电脑（同名用户）       │
│                                                      │
└──────────────────────────────────────────────────────┘


┌─── 常用操作 ────────────────────────────────────────┐
│                                                      │
│  start.bat    启动本机 agent（之后用这个）             │
│  stop.bat     停止本机 agent                          │
│  setup.bat    重新配置（云端换地址、密钥换了再跑）      │
│                                                      │
└──────────────────────────────────────────────────────┘


┌─── 常见问题 ────────────────────────────────────────┐
│                                                      │
│  问：浏览器报 "Failed to fetch"                       │
│  答：本机 agent 没启动；双击 start.bat                │
│                                                      │
│  问：浏览器报 "wrong_user"                            │
│  答：本机绑定的用户名跟你登录的不一致；重跑 setup.bat  │
│                                                      │
│  问：浏览器报 "agent_not_bound"                       │
│  答：还没绑定；双击 setup.bat                         │
│                                                      │
│  问：start.bat 闪一下就没了                           │
│  答：可能 .venv 没装好。重跑 setup.bat               │
│                                                      │
│  问：LLM 调用慢 / 失败                                │
│  答：在浏览器"系统配置"里填 LLM API key 并测试连接     │
│                                                      │
└──────────────────────────────────────────────────────┘


有问题联系管理员；如果是技术问题贴日志会更快：
  日志在 C:\Users\<你>\.handynotes\<云端用户名>\mjq.log
