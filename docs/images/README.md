## 截图技巧

- **截图工具**：Windows 自带 `Win+Shift+S`（剪切板）；Snipaste（更顺手）
- **打码**：含敏感信息（真实姓名 / 真案件名）的截图，先用截图工具的"马赛克"功能糊一下，或直接拿测试数据。
- **格式**：推荐 PNG（截图本来就大量是文字 + 简色块，PNG 比 JPG 清晰）
- **压缩**：单图 > 500 KB 时建议在 https://tinypng.com 压一下，免得 markdown 加载慢

## 凑齐截图后

把整个 `docs/` 文件夹丢给最终用户，或者：

- 转成 PDF：用 VS Code 装 "Markdown PDF" 插件，或 https://md-to-pdf.fly.dev
- 挂上服务器：复制到 `/opt/mjq-handynotes/public-downloads/`，文件名比如 `用户使用手册.pdf`，
  然后在浏览器登录页加一个"使用说明"链接（同 `/downloads/` 那个 nginx 配置）。
